Selector Hover ぁ

`Selector Hover` is used to select elements when the user hovers over them with a pointing device, such as a mouse. It is commonly used to create interactive effects on web pages, such as changing the color of a button when the user hovers over it.

Example

To change the background color of a button when the user hovers over it: 

    
      button {
        background-color: #008cba;
        color: white;
        padding: 10px 20px;
        border: none;
        cursor: pointer;
        border-radius: 10px;
      }
      button:hover {
        color: #ff0000;
        transition: all 0.3s ease;
      }
