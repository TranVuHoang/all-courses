3.16 - Selector: First and Last Child

`Selector: First and Last Child` is used to select the first and last child elements of a parent element. It allows you to apply styles to the first and last child elements of a container, which can be useful for creating specific layouts or emphasizing certain content.

Example

To select the first and last child elements of a container: 
    
      .container:first-child {
        color: blue;
      }
      .container:last-child {
        color: red;
      }
