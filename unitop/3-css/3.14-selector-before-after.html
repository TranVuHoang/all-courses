Selector Before and After

`Selector Before` and `Selector After` are used to insert content before or after an element. They are commonly used to add decorative elements or to create complex layouts without adding extra HTML elements.

Example

To insert content before and after a paragraph: 
      
      p::before {
        content: "Before: ";
        color: red;
      }
      p::after {
        content: " :After";
        color: blue;
      }

This is a paragraph.
